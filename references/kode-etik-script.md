## 0. Apa ini / apa BUKAN ini

| Ini (guidance) | Bukan ini (dilarang) |
|----------------|----------------------|
| Filter **sebelum** milih angle & closing | Tema deck: “true thrift vs false thrift” |
| Cegah insight asal / khotbah hemat / sanksi moral | Headline pakai label `TRUE SPEND` / `FALSE THRIFT` / `KODE ETIK` |
| Putusin: hemat di sini? ganti di sini? diam? | Pack 10 script yang semua “alat rusak / hemat palsu / gunting” |
| Invisible — pembaca ngerasa cerita dompet | Pembaca ngerasa dikasih kuliah filsafat hemat |

**Produk yang dibaca orang = insight ber-receipt + suara human.**  
Kode etik **tidak** boleh kelihatan sebagai isi. Kalau deck lo kedengeran kayak PowerPoint kategori uang, lo gagal.

**Satu kalimat filter:**

> Hemat di tempat yang harus hemat. Bayar/ganti di tempat yang harus.  
> *(Gunting tumpul + pakai tiap hari = murah palsu — itu **contoh putusan**, bukan judul batch.)*

## 1. Kapan pakai filter ini

**Pakai** saat:

- Milih angle (boleh digali / stop)
- Nulis closing & saran aksi
- Ragu: “apakah ini cuma bilang hemat doang?”

**Jangan pakai** sebagai:

- Outline slide (“slide 3 = false spend, slide 4 = true thrift”)
- Kicker / topic string yang meta
- Alasan bikin 10 deck mirip dengan label etik beda-beda

---

## 2. Empat kategori — di kepala, bukan di copy

Internal vocabulary (boleh dipikir, **jangan dijadikan headline/body default**):

| Internal | Arti | Efek ke script |
|----------|------|----------------|
| True thrift | Cut yang gak dipakai | Closing bisa: cancel, ambil sendiri, stop nambah |
| False thrift | Nahan duit → rugi lebih gede | Jangan saranin; kalau muncul di hidup reader, **bongkar** sebagai jebakan lewat cerita+angka |
| True spend | Bayar/ganti yang frekuensi+konsekuensi nuntut | Boleh saranin ganti/bayar full **dalam bahasa manusia**, bukan label |
| False spend | Keluar duit tanpa pos/pakai | Bongkar mekanisme + angka; stop |

**Dilarang di plan.json (kecuali kutip orang / meta-review internal):**

```
TRUE THRIFT / FALSE THRIFT / TRUE SPEND / FALSE SPEND
KODE ETIK / HEMAT PALSU (sebagai judul berulang)
“Ini contoh true spend…”
```

Tulis: *“Ganti mouse yang dobel-klik — 8 jam/hari.”*  
Jangan: *“TRUE SPEND: ganti mouse.”*

## 3. Cegah “asal insight” (tujuan asli owner)

Asal insight = deck yang:

1. Gak punya **lived open** (situasi lo/you + detail)  
2. Gak punya **receipt baru** per slide (angka/nama/mekanisme)  
3. Closing generik (“hemat ya”, “catat pengeluaran”, “download app”)  
4. Moral theater / misery cult / “jangan beli apa-apa” di semua topik  
5. Sunk-cost virtue (“udah bayar → wajib dipaksa pakai”)  
6. Clone skeleton (260 workdays, staccato `X. Y.` mati, 10× “alat rusak”)

**Sebelum nulis**, angle harus jawab:

1. **Situasi spesifik apa** yang reader pernah alami minggu ini?  
2. **Satu mekanisme / angka** yang bikin “oh, gitu ya”?  
3. **Satu aksi** yang etis *untuk situasi itu* (bukan slogan hemat universal)?

Gagal 1–3 → **ganti angle**, jangan generate 8 slide kosong-berirama.

---

## 4. Uji frekuensi × konsekuensi (internal)

Untuk **benda/langganan/utang di dalam cerita** (bukan untuk bikin deck tentang uji ini):

1. Seberapa sering dipakai?  
2. Kalau jelek/telat, rugi apa?  
3. 90 hari: ganti/bayar full vs nerusin — mana lebih murah?

| Frekuensi | Kondisi | Putusan (internal) |
|-----------|---------|---------------------|
| Harian / kerja | Rusak, tidak aman | condong **ganti/bayar** |
| Jarang / identitas | 0 pakai | condong **cancel** |
| Auto-debit | Lupa | alarm + cancel |
| Bunga/denda | Minimum doang | kejar pokok/full |
| Convenience | Markup jelas | sebut trade-off; jangan fatwa “haram order” |

---

## 5. Diversitas pack (anti-redundant)

Satu pack / wave:

- **Maks 1** deck yang angle-nya “alat harian rusak / ganti vs hemat palsu” (gunting/helm/sepatu/mouse — pilih **satu** kalau perlu, jangan empat).  
- Campur domain: fee app, utang, langganan, gaji, dapur, RS, cicilan, scam, pajak, transfer, kerja…  
- Dua deck boleh “hemat” asalkan **mekanisme & objek uang beda** (GoFood markup ≠ flash sale ≠ top-up game).  
- Kalau 5+ cover kedengeran satu skeleton moral → **STOP wave**, rewrite angles.

Gold diversity bar: forensic ID `5301–5320` (GoFood, denda, admin bank, e-wallet, streaming, scam, KPR/SLIK, umroh…) — etik **di belakang**, insight **di depan**.
