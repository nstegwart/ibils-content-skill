### Body law (both languages)

1. Open with **you / lo / gue** situation — never source/date/publisher first.
2. Source mid-sentence or after the human beat.
3. One **new** fact or mechanism per content slide (Receipt Law — `lint-plan.js`).
4. ≥50% content slides carry a figure.
5. Max one dense "receipt tower" mid-deck; rest stay spoken.
6. Closing = ultra-short punch for **this** story (not generic CTA).
7. Jargon most people don't know → **one-breath gloss** on first use  
   (OTR = harga siap jalan; SLIK = rapor utang OJK; PPIU = izin travel umroh; KYC = verifikasi KTP…).

### Banlist — hard FAIL (`lint-voice.js`)

**ID:** `otak…` metaphors · `dada lo naik` default · `buffer` → use cadangan/sisa/dana darurat · English welds (`framing UI`, `reward app`) · source-first body · cover with no money object and no lo/gue/wallet situation · **cover GUE+LO mash** (satu POV saja) · **wrong collocations** di bank `id-collocation-ban.json` (contoh: `tiap gaji` → `tiap gajian`).

**EN:** body opens `According to` / `Per BLS` / `Deloitte's` / `FinanceBuzz ran` · cover `$X … 260 WORKDAYS … TRIP YOU NEVER` · uncontracted press-release as default.

**Both:** empty payoffs, balanced triplets, unlock/seamless/game-changer, mid-deck deferral (`tips?` headline trap), listicle headline starting with a bare digit on content slides.

### Owner nit → never again (loop wajib)

1. Owner bilang "WTF / salah bahasa / AI slop" → **bukan** cuma rewrite satu item.
2. Kalau polanya **regex-able** (kolokasi, scaffold cover, jargon EN di ID): append rule ke `references/id-collocation-ban.json` (atau hard rule di `lint-voice.js` / `lint-quality.js`).
3. Gate triple: `lint-plan` + `lint-voice` + `lint-quality` — FAIL = **tidak ship** (`write-batch.sh`).
4. Kalau polanya **ear-only** (staccato yang lolos telegram test, "rasa AI"): tambah **few-shot bad** di `gold-fewshot-id.md` + contoh di `writing-research-id.md` — AI next wave harus liat counterexample.

## 3. Spine of a good script (forensic or insight)

```
1 COVER     lived open + number/name + wrong belief / twist
2–3         situation deepens (lo/you still in the room)
mid         mechanism + named receipt (source tucked, not leading)
late        wallet so-what (90-day math, pos, calendar)
last-1      action that passes KODE ETIK (right thrift / right spend)
CLOSING     2–4 words, this story only
```

Self-check before lint:

1. Screenshot-send test — would a tired worker/parent forward this?
2. Cover names product/situation + human mistake?
3. Body 1 = human beat, not citation?
4. Zero otak / 260-farm / source-first stack?
5. If sources[] deleted, does the emotional story still stand?
6. **Ethics (kode-etik-script.md E1–E7):** right thrift **and** right spend? gunting-tumpul risk?
7. Surface + jargon gloss + screenshot-send?

Fail any → rewrite. Do not ship.
